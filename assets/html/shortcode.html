Shortcodes



Return to Docs Index

The Shortcodes listed here are to aid in the creation of custom formatting that comes standard with your site. In using these Shortcodes, HTML tags can be avoided when entering content in WordPress. All of these Shortcodes can be accessed through the Shortcodes drop-down list on the WordPress Visual editor, this section is simply for reference.

Note: If including text surrounded in double quotes, change the opening and closing quotes to single quotes. Ex: title="The Title Here" becomes title='The "Title" Here'.

[hero title="" author="" subtitle="" vimeo_id=""]

[hero] generates the Hero section for Feature articles and requires a Title, Author, Subtitle and Vimeo ID. The Vimeo ID is the numeric value at the end of a Vimeo URL (ex: for the URL http://vimeo.com/90667610, the value would be vimeo_id="90667610").

[small-caps]Text Here[/small-caps]

Surround text that needs to by styled as small caps.

[blockquote align=""]Text Here[/blockquote]

[blockquote] should be used instead of WordPress's default blockquote code. The Align value can accept 'right', 'left', 'center', and defaults to 'center'.

[hr-break]
[hr-thick]

Two different styles of Horizontal Rule are available.

[h3]Text Here[/h3]
[h4]Text Here[/h4]

Use [h3] and [h4] for primary and secondary headings on pages.

[video align="" vimeo_id=""]Image Here[/video]

[video] requires Align and Vimeo ID parameters. Align accepts 'left' and 'right', and the Vimeo ID follows the same guidelines as in the [hero] tag above. Insert the [video] Shortcode first, then delete the "Image Here" text and insert an image in its place using the Add Media button.

[email]name@genomemag.com[/email]

Wrap all email addresses in the [email] tag in order to create a mailto link and to hide email addresses from spam bots.

[contributor author="" posts="" position="first"]
[contributor author="" posts=""]
[contributor author="" posts="" position="last"]

[contributor] uses Author, Posts and Position attributes. Author requires the Username found under a User's profile. Posts accepts a numeric value for the number of posts to display. The first and last contributor must have the Position tag as in the illustration above. This Shortcode is used on the Contributors page.

[person name="" title="" link="#" position="first"]
[person name="" title="" link="#"]
[person name="" title="" link="#" position="last"]

This Shortcode is used on the About us page and accepts a Name, Title, Link and Position attribute. The Link may be to an external site. The Position attribute follows the same guidelines as the [contributor] Shortcode described above.

[contact name="" phone=""]

[contact] is used on the Media Kit page and requires a Name and Phone attribute.

[vcard name="" address1="" address2="" address3=""]

[vcard] is used on the Contact Us page. It accepts the Name attribute, and each Address attribute is displayed on a new line.
