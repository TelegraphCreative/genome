Horizontal Rules

The handy <hr/> tag: useful for creating breaks in content. 




  <hr class="partial">
              



  <hr>
              



  <hr class="thick">
              



  <hr class="thick partial">
              



  <hr class="brick">


Get fancy and add background-color classes (find more in utilities): 



  <hr class="primary-bg thick partial">


On top of these, you can create custom hr tags like this one:




First, use the .article-break class as a foundation to apply the basic rules you'll need. Then, add a custom class describing your new hr: in this case, .health-journal-hr.


HTML

  <hr class="article-break health-journal-hr">


In your CSS, target the new class you created and specify a background image of your choosing. Viola! A completely custom hr created with a background image of your own.


CSS

  .health-journal-hr {
      background-image: url("../img/ornaments/article-break-health-journal-icon-retina.png") !important;
  }

Blockquotes

Sometimes other people say smart things, and you may want to mention that through a blockquote callout. We've got you covered.



“We cannot live only for ourselves. A thousand fibers connect us with our fellow men; and among those fibers, as sympathetic threads, our actions run as causes, and they come back to us as effects.”
Herman Mellville 


  <blockquote cite="http://url-here-if-there-is-one.com">
    “We cannot live only for ourselves. A thousand fibers connect us with our fellow men; and among those fibers, as sympathetic threads, our actions run as causes, and they come back to us as effects.”
  </blockquote>
  <cite>Herman Mellville</cite>

Print Styles

Genome's main stylesheet includes print styles developed to present the printed version of the site in an elegant way. These are activated when you print through a media query. Here's some examples of how a GenomeMag.com looks when printed out:






In addition to these print styles, genome-main.css includes some simple classes you can use to control elements printing, or not printing. Simply attach .show-for-print to an element to only show when printing, and .hide-for-print to hide something when printing.
