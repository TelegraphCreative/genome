HTML Structure

Featured articles break down into two major parts: first, the article title is contained in .hero-wrap, then, the main content (i.e. article copy, video heros, and edge-to-edge slideshows) falls within #rte-target. Read time is estimated by using #rte-target, so it's essential this wraps the copy to generate the correct reading time estimate.


TITLE HERO & RTE-TARGET

  <div class="hero-wrap">
    <header class='hero hero-featured-article background-size-full center hero-tall'>
      <div class='story-cover-content single-col-layout'>
        <div class='align-middle'>
          <div class='align-block'>
            <a class='cover-content-inner' data-0='top:0px;' data-500='top:-40px;' href='https://www.google.com/'>
              <h2>
                [FEATURE HEADING]
              </h2>
              <hr class='white-bg partial'>
              <p class='sans lead'>
                [FEATURE SUB-HEADING]
              </p>
              <div class='author-attrib text-meta-highlight tertiary-text-color'>[AUTHOR NAME]</div>
              <div class='content-row'>
                <div class='arrow-down' data-icon='E' href=''></div>
              </div>
            </a>
          </div>
        </div>
      </div>
      <div class='story-cover-image' data-0='opacity:1;' data-140='opacity:0;'></div>
      <div class='story-cover-image-transition'></div>
    </header>
  </div>

  <div id='rte-target'>
    [MAIN ARTICLE CONTENT (copy, heros, slideshows, etc)]
  </div>

Using Skrollr.js

Skrollr.js is a parallax scrolling library. Use it create elegant transitions in a snap. Below is an example of how to transition a block of content upward. For more examples and ideas, check out the Wiki on Github or reference the markup in this Parallax example.

The Basics
Using skrollr.js, you can transition any CSS attribute. Enter your starting value in data-0='' and ending value in data-500=''. The number following data- specifies the duration of the transition. In this example, .cover-content-inner will start at 0px and move upward by -40px for a duration 500px scrolled.


HTML

  <a class='cover-content-inner' data-0='top:0px;' data-500='top:-40px;' href='https://www.google.com/'>
    <h2>
      [FEATURE HEADING]
    </h2>
    <hr class='white-bg partial'>
    <p class='sans lead'>
      [FEATURE SUB-HEADING]
    </p>
    <div class='author-attrib text-meta-highlight tertiary-text-color'>[AUTHOR NAME]</div>
    <div class='content-row'>
      <div class='arrow-down' data-icon='E' href=''></div>
    </div>
  </a>
