Inserting Media

Images

All images and galleies can be added through WordPress's default Add Media button (located directly above the body text editor). There are three important fields for images:

  • Caption - this field is displayed below images. It is important to note that changing this field will change all instance of the caption for the photo throughout the site.
  • Alignment - This determines whether your photo sits in the body content ('Center' and 'None'), or whether it is pulled left or right.
  • Link To - This should either be set to 'None' for no link, or 'Media File' to display a link to enlarge the image. Support for Custom URL and Attachment Page is not supported.

Galleries

Galleries are added by clicking on the Create Gallery link at the top left of the page after clicking the Add Media button. Simply select two or more images to insert a gallery and click the 'Create a new gallery' button. On the next screen you will have the option to change the Columns field. Setting Columns to 1 will display a full page rotating gallery. Choosing anything above 1 will display the first image and caption of the gallery in the body text with a link to a pop-up gallery.

Videos

Videos are added using either the [hero] or [video] Shortcode. Please refer to the Shortcode documentation to learn more.
